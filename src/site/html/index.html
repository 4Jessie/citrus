Features

Citrus is able to perform high complex integration tests with great validation capabilities. See the complete feature list for details.

  • Mocks and simulators: out-of-the-box client and server simulation
  • Test automation: execute integration tests with continuous builds
  • Protocol support: HTTP, JMS, TCP/IP, REST, SOAP, XML, JSON, and more
  • Validation/Verification: validate message headers and body in each test step
